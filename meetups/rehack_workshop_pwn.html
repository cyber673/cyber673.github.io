> cyber673 x RE:HACK: Basic in Pwn (workshop)

Date: 21 June 2025

Time: 2PM - 6PM

Location: Level 1, Brunei Innovation Lab, KHUB Building, Anggerek

Irfan Haziq (capang)
This workshop provides hands-on insights into low-level program behavior and exploitation techniques. Participants will begin by understanding how a program runs, including the structure of the stack during function calls and exits. It then explores what happens during a buffer overflow, followed by an introduction to Return Oriented Programming (ROP) which is an essential technique in binary exploitation.
